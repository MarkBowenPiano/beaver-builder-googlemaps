Add a new module to create a Google Maps in Beaver Builder.


== Description ==

Map options
1. Locate marker by Lattitude and Longitude
2. Zoom level
3. Map height
4. Custom marker
5. Custom text (wysiwyg) inside marker popup


== Installation ==

1. /!\ Need Beaver Builder to work.
2. Install and activate Beaver Builder Google Maps Module either via the WordPress plugin directory, or by uploading the files to your server at wp-content/plugins.
3. Beaver Builder Google Maps Module will now be available to use for any of the post types you chose.
